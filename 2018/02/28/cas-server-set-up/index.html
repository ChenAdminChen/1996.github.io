cas的学习

下载基于5.2x版本的cas服务器

    下载地址

cas.properties文件配置

    在项目的cas-overlay-template\etc\cas\config文件下需要配置cas.properties内相关的配置内容。

    以下是相关配置文件

注册服务

service.json

配置pom.xml

         在cas-overlay-template项目下存在一个pom.xml文件，需要添加依赖包后，才会开启相应的功能，否则只存在一部分基础功能

SSL认证

生成私钥

    利用jdk自身的keytool生成相应的数字一个证书,生成证书到D:\localhost.keystore

生成公钥

    首先使用localhost.keystore导出数字证书（公钥）到D:\localhost.cer

将其配置到jdk内

    因为CAS client需要使用该证书进行验证，需要将证书导入到JDK中

打包运行

    打包成功后会,项目内会出现target文件内存在cas.war的包

下面的步骤不需要使用

将cas.war放入tomact容器

    将cas.war放入D:\apache-tomcat-8.0.30\webapps内

与tomact同级创建ect文件

    将项目中的ect文件全部复制到与tomact同级的磁盘存储,在该文件下只改变cas.properties内的内容时,不需要将项目重新打包,只需要将tomact重启就可生效配置文件

配置tomact的,认证SSL ,其中这一步可不执行,跳转到

    配置D:\apache-tomcat-8.0.30\conf下的server.xml文件,将下面的内容添加进出,认证SSL,其keystoreFile所指向的地方:为私钥存放的地方

    将其内容配置好后,启动tomcat,访问地址:https://127.0.0.1:8443/cas/login
