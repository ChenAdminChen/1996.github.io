触发器可能会导致主键重复的错误，原因是A表中的数据由此触发器插入，但数据导入脚本也导入了已有数据，导致冲突。

因此，在触发器中增加对用户自定义变量 @disable_triggers 的检查，此变量为null时才执行动作。当需要禁止触发器动作时，设置此变量为1即可。
